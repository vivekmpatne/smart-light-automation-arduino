# Smart Light Automation using Arduino

A simple smart automation project built using Arduino UNO, relay module, sound sensor, and Python voice commands.

## Features
- Clap / sound controlled switching
- Voice command control
- Relay-based output switching
- Real-time serial communication

## Tech Used
- Arduino UNO
- Arduino IDE (C++)
- Python
- Relay Module
- Sound Sensor

## Learning Outcomes
- Hardware-software integration
- Serial communication
- Relay control logic
- Sensor debugging
- Basic automation concepts
